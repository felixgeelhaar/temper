# Exercise Packs

Temper includes 41+ exercises across four languages.

## Available Packs

| Pack | Language | Exercises | Difficulty |
|------|----------|-----------|------------|
| go-v1 | Go | 15 | Beginner-Advanced |
| python-v1 | Python | 14 | Beginner-Advanced |
| typescript-v1 | TypeScript | 11 | Beginner-Advanced |
| rust-v1 | Rust | 13 | Beginner-Advanced |

## Listing Exercises

```bash
# List all packs
temper exercise list --packs

# List exercises in a pack
temper exercise list --pack go-v1
```

## Starting an Exercise

```bash
temper exercise start go-v1/basics/hello-world
```
