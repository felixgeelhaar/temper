# Progress Tracking

Temper tracks your learning journey with evidence-based metrics.

## Viewing Statistics

```bash
# Overview
temper stats

# Skill progression
temper stats skills

# Error patterns
temper stats errors

# Hint dependency trend
temper stats trend
```

## Metrics Tracked

### Skills
- Topics practiced
- Success rate by topic
- Improvement over time

### Errors
- Common error patterns
- Frequency by type
- Reduction over time

### Hint Dependency
- Hints per session
- Escalation rate
- Independence growth

## Appreciation

Temper provides calm, evidence-based feedback:
- "Error rate for 'nil pointer' reduced from 5 to 1"
- "Completed 3 exercises without L3 assistance"
- "Hint dependency down 40% this week"

No gamification. Just honest progress recognition.
